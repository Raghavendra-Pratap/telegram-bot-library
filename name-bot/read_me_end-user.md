## 📖 How to Use

### Step 1: Add the Bot
1. Open your Telegram channel or group
2. Go to **Settings** → **Administrators**
3. Click **Add Administrator**
4. Search for `@uploadwithcaption_bot`
5. Add the bot as an administrator

### Step 2: Set Permissions
When adding the bot, make sure to enable:
- ✅ **Edit messages** (Required - this is how the bot adds captions)

**Optional for groups:**
- ✅ **Delete messages** (Helps in some group scenarios)

### Step 3: Start Uploading!
That's it! Just upload files to your channel or group:
- Upload any file (video, document, photo, audio, etc.)
- The bot automatically detects the file
- Caption is added within seconds
- Done! No commands needed

## ⚙️ How It Works

1. **You upload a file** to your channel/group
2. **Bot detects the file** automatically
3. **Bot extracts the filename** from the file
4. **Bot adds the filename as caption** to your message
5. **Done!** Your file now has a caption

**Important:** The bot edits your message to add the caption - it doesn't re-upload the file. This means:
- ✅ No duplicate uploads
- ✅ No bandwidth waste
- ✅ Fast and efficient
- ✅ Original upload stays intact

## ❓ Frequently Asked Questions

### Q: Do I need to run any commands?
**A:** No! After initial setup, just upload files. The bot works automatically.

### Q: Will the bot overwrite my existing captions?
**A:** No. If a file already has a caption, the bot won't add another one.

### Q: Does it work in groups?
**A:** Yes! Works in channels, groups, and supergroups.

### Q: Can I use it in multiple channels?
**A:** Yes! Add the bot to as many channels/groups as you want.

### Q: What if the bot doesn't add a caption?
**A:** Check that:
- Bot is added as administrator

- File supports captions (video notes and stickers don't)

### Q: Does it work with forwarded files?
**A:** Yes! The bot can add captions to forwarded files too.

### Q: Is it free to use?
**A:** Yes! The bot is completely free to use.
